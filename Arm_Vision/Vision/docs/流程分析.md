# 自动充电机器人视觉引导流程

## 前置准备（一次性）

- YOLO11 模型训练（识别 `charging_port`）
- STP 数模 → 生成 ICP 模板点云
    - `/home/admin123/miniconda3/envs/yolo_env/bin/python step_to_pcd.py`
- 手眼标定（相机与机械臂末端的相对位置关系 T_cam2gripper）
- 示教"搜索位"（机械臂出发点，相机能大致看到充电座）

---

## 运行时流程

### 第一步：运动到搜索位

机械臂移动到预设的搜索位，确保相机能看到充电座。

### 第二步：粗位姿估计

- YOLO11 检测充电座（仅需 `charging_port` 一个类别）
- 结合深度图反投影，计算充电座粗略的 6DoF 位姿
- 精度：平移 ±3~5mm，旋转 ±5~8°

执行：`bash run_pose_estimation.sh`

### 第三步：运动到标准拍摄位

根据粗位姿，引导机械臂移动到标准拍摄位：

- 距充电座约 50cm
- 相机垂直正对充电座
- 充电座位于画面正中

| 参数 | 值 | 含义 |
|------|------|------|
| X | 0mm | 充电座在画面正中（不偏左右）|
| Y | 0mm | 充电座在画面正中（不偏上下）|
| Z | 500mm | 距离充电座 500mm |
| Rx | 0° | 相机不上下倾斜 |
| Ry | 0° | 相机不左右偏转 |

### 第四步：ICP 精确定位

- 打开相机，拍一帧
- YOLO 找 bbox（确定充电口区域，以便裁剪点云）
- 加载 STEP 模板点云
- 以标准拍摄位（X=0, Y=0, Z=500mm）为初始值，跑 ICP
- 输出精确 6DoF 位姿，精度：平移 ±0.5mm，旋转 ±1°

执行：`bash run_icp_refine.sh`

### 第五步：坐标系转换

```
精确位姿（相机坐标系）
    × 手眼标定矩阵
    → 充电座位姿（机械臂基座坐标系）
```

### 第六步：插枪

- 机械臂移动到充电座正前方预备位
- 切换力控模式，执行插枪动作
- 检测到插入力反馈，确认插接成功

## 算法原理

### 粗位姿（pose_estimation.py）

```
YOLO 检测 bbox
    ↓
bbox 区域深度中值 → 反投影 → 3D 坐标（X, Y, Z）
    ↓
RANSAC 平面拟合 → 法向量 → 充电座朝向（Rx, Ry）
    ↓
旋转矩阵（世界Y轴 × 法向量）→ 6DoF 位姿
```

### 精确位姿（icp_refine.py）

```
标准拍摄位初始变换（T = I, Z=0.5m）
    ↓
模板点云（STEP 生成）vs 当前帧点云
    ↓
Point-to-Plane ICP 精对准
    ↓
精确 6DoF 位姿
```
